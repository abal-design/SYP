# Face Attendance System (Team Scaffold)

This repository is a collaboration-ready scaffold based on the structure of the original Face Attendance System project.

## Monorepo Layout

- client: Frontend app (Vite/React style layout)
- server: Backend API service (Node.js/Express style layout)
- .github: Collaboration templates and standards

## Quick Start

1. Clone this repository.
2. Open the workspace in VS Code.
3. Install dependencies inside each app:
   - client: npm install
   - server: npm install
4. Create local env files from examples.
5. Start development servers.

## Team Collaboration

- Use feature branches for every change.
- Open Pull Requests for all merges to main.
- Follow CONTRIBUTING.md for code and review workflow.
- Use issue templates when reporting bugs or requesting features.

## Suggested Branch Naming

- feat/<short-feature-name>
- fix/<short-bug-name>
- chore/<task-name>
